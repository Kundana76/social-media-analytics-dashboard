# Social Media Engagement Analytics Dashboard 📊

An interactive **Social Media Analytics Dashboard** built using **HTML, CSS, JavaScript, and Chart.js** to analyze and visualize social media engagement metrics such as likes, comments, shares, reach, and follower growth.

---

## 🚀 Live Demo

After deploying with GitHub Pages, your dashboard will be available at:

https://kundana76.github.io/social-media-analytics-dashboard/

---

## 📌 Features

* Interactive KPI cards for key metrics
* Engagement rate trend analysis
* Post type performance comparison
* Follower growth visualization
* Top performing posts ranking
* Hashtag analysis dashboard
* Reach vs impressions analytics
* Virality score tracking
* Dynamic filters for post type and date range

---

## 📊 Metrics Included

The dashboard analyzes the following metrics:

* Total Posts
* Likes
* Comments
* Shares
* Saves
* Engagement Rate

## 📈 Calculated Metrics

The dashboard calculates the following performance indicators:

**Engagement Rate**

```
(Likes + Comments + Shares + Saves) / Reach × 100
```

**Virality Score**

```
(Shares + Saves) / Impressions × 100
```

**Follower Growth Rate**

```
Followers_Gained / Reach × 100
```
